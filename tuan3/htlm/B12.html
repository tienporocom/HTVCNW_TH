Cascading Style Sheets Book

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language. Although most often used to change the style of web pages and user interfaces vffitten in HAL and XHTN'IL, the language can be applied to any kind document, including plain XNfL, SVG and XIJL. Along with HTNIL and JavaScript, CSS is a comerstone technology used by most websites to create visually engaging webpages, user interfaces for web applications, and user interfaces for many mobile applications.
